Exporting

To export a dataset:

  1. Go to the user page by either clicking on user page in the navbar or select data in side bar

  2. Then select the dataset that you want from the list

  3. From the sidebar in the dataset page, choose the file type (CSV or Xlsx) and click on "Export" button.

Tagging a Dataset

Creating Tags

To create a tag to use on your dataset:

  1. Log in to your account if you are not already logged in.

  2. Navigate to your user page.

  3. Select the dataset you want to create a tag for.

  4. On the Dataset page, enter the text for the tag in the large text area in the upper left corner.

  5. Click the “+ Add” button to add the tag.

  6. You can now use this tag on this dataset.


    This is a screenshot of the tagging page


Deleting Tags

To delete a tag from the dataset:

  1. Log in to your account if you are not already logged in.

  2. Navigate to your user page.

  3. Select the dataset.

  4. Click on the "x" next to the tag you want to remove from the list of tags. This will remove the tag from that response.


Adding Tags to a Response

To assign tags to a response:

  1. Log in to your account if you are not already logged in.

  2. Navigate to your user page.

  3. Select the dataset.

  4. If the tag you want to use is not already created, create it.

  5. Click and drag the tag you want from the list of tags in the lower left corner to the response you want to tag. The tag should now be displayed in the list of tags for that response.


Removing Tags from a Response

To remove a tag from a response:

  1. Log in to your account if you are not already logged in.

  2. Navigate to your user page.

  3. Select the dataset.

  4. Click on the "x" next to the tag you want to remove from the response. This will remove the tag from that response.
